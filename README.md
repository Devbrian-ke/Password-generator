
# Password Generator

A simple, lightweight desktop application for generating secure random passwords — built with Python and Tkinter.

---

## Overview

**Password Generator** is a GUI-based desktop tool that lets you instantly create strong, random passwords of any length. It combines uppercase and lowercase letters, digits, and special characters to maximize password entropy — no internet connection required.

---

##  Features

- Generates passwords using letters, digits, and special characters
-  Customizable password length
- Clean, minimal desktop GUI (no browser needed)
-  Instant generation with one click
-  Fully offline — your passwords never leave your machine

---

##Requirements

- Python **3.6+**
- `tkinter` (included with most standard Python installations)

> **Note:** On some Linux distributions, Tkinter may need to be installed separately (see below).

---

##  Installation

### 1. Clone or Download the Project

```bash
git clone https://github.com/your-username/password-generator.git
cd password-generator
```

Or simply download the `password_generator.py` file directly.

### 2. Verify Python is Installed

```bash
python --version
```

If Python is not installed, download it from [python.org](https://www.python.org/downloads/).

### 3. Install Tkinter (if needed)

Tkinter comes bundled with Python on **Windows** and **macOS**. On **Linux**, install it with:

```bash
# Ubuntu / Debian
sudo apt-get install python3-tk

# Fedora
sudo dnf install python3-tkinter

# Arch Linux
sudo pacman -S tk
```

---

## Usage

Run the application from your terminal:

```bash
python password_generator.py
```

### Steps:

1. **Enter a length** — Type the desired number of characters in the input field.
2. **Click "Generate Password"** — A new random password will appear below the button.
3. **Copy the password** — Highlight and copy it for use wherever you need it.

##  How It Works

The app uses Python's built-in `random` and `string` modules:

| Component | Source |
|-----------|--------|
| Uppercase & lowercase letters | `string.ascii_letters` |
| Digits (0–9) | `string.digits` |
| Special characters | `string.punctuation` |

A password is generated by randomly selecting characters from the combined pool, repeated for the specified length.
